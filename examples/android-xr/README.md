# Embardiment for Android (XR) Example

This is the example Unity project for Embardiment for Android (XR).

You can get a copy of this project by downloading this repository and browsing to `./examples/android-xr`.  If you want, you may move and rename this directory, and you may delete all other files from your copy of the repo.

To open in Unity Hub, select **Add > Add project from disk** and select this project.

More information about this project can be found [in the Project's README.md](../../README.md).
